# 🌳 Decision Tree Classification Project

This repository contains a Machine Learning project implementing the Decision Tree algorithm for classification tasks.

---

## 📌 Project Overview

Decision Tree is a supervised machine learning algorithm used for classification and regression. It works by splitting the dataset into branches based on feature values, forming a tree-like structure.

This project demonstrates:

* Building a Decision Tree model
* Training and testing the model
* Evaluating model performance

---

## 📂 Dataset

The dataset used in this project contains labeled data used for training and evaluating the Decision Tree model.

---

## 📚 Steps Performed

### 🔹 Data Preprocessing

* Handling missing values
* Encoding categorical variables
* Preparing training and testing datasets

### 🔹 Model Building

* Applying Decision Tree algorithm
* Training the model

### 🔹 Model Evaluation

* Predicting test results
* Confusion Matrix
* Accuracy calculation

### 🔹 Model Interpretation

* Understanding tree structure
* Feature importance analysis
