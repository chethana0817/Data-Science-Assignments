# 🔄 Data Transformation & Preprocessing

This repository contains a data preprocessing and transformation project using Python. The focus is on preparing raw data for analysis and machine learning.

---

📌 Project Overview

Data preprocessing is a crucial step in data science that ensures data is clean, structured, and ready for analysis or model building.

This project demonstrates how to:

* Clean raw data
* Handle missing values
* Transform features
* Prepare datasets for machine learning

---

📚 Tasks Performed

### 🔹 Data Cleaning

* Handling missing values
* Removing duplicates
* Fixing inconsistent data

### 🔹 Data Transformation

* Feature scaling (Normalization / Standardization)
* Encoding categorical variables
* Data formatting

### 🔹 Data Preparation

* Splitting dataset
* Preparing input features and target variables

---

🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Jupyter Notebook

---

▶️ How to Run

1. Install required libraries:

   ```bash
   pip install numpy pandas scikit-learn jupyter
   ```
2. Launch Jupyter Notebook:

   ```bash
   jupyter notebook
   ```
3. Open the notebook and run all cells
