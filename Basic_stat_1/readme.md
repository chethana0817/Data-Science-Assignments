📊 Basics of Statistics - 1

This repository contains a Python-based assignment focused on fundamental statistical concepts used in data analysis and data science.

---

📌 Project Overview

This project introduces essential statistical methods required to understand and analyze data effectively. It demonstrates how statistical concepts can be applied using Python.

---

📚 Topics Covered

### 🔹 Descriptive Statistics

* Mean
* Median
* Mode
* Variance
* Standard Deviation

### 🔹 Probability Basics

* Fundamental probability concepts
* Simple probability calculations

### 🔹 Data Analysis Concepts

* Understanding datasets
* Summarizing data
* Drawing basic insights

---

🛠️ Technologies Used

* Python
* NumPy
* Pandas
* Jupyter Notebook

---

▶️ How to Run

1. Install required libraries:

   ```bash
   pip install numpy pandas jupyter
   ```
2. Launch Jupyter Notebook:

   ```bash
   jupyter notebook
   ```
3. Open the notebook file and run all cells
