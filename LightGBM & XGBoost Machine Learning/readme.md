# 🚀 LightGBM & XGBoost Machine Learning Project

This repository contains a Machine Learning project implementing advanced ensemble algorithms: LightGBM and XGBoost for predictive modeling.

---

## 📌 Project Overview

LightGBM and XGBoost are powerful gradient boosting algorithms widely used in industry for their high performance and efficiency.

This project demonstrates:

* Building advanced ML models
* Training and evaluating boosting algorithms
* Comparing model performance

---

## 📂 Dataset

The dataset used in this project contains multiple features used to train and evaluate predictive models.

---

## 📚 Steps Performed

### 🔹 Data Preprocessing

* Handling missing values
* Encoding categorical variables
* Feature selection and preparation

### 🔹 Model Building

* Training LightGBM model
* Training XGBoost model

### 🔹 Model Evaluation

* Predicting results
* Accuracy and performance comparison
* Model evaluation metrics

### 🔹 Model Comparison

* Comparing LightGBM vs XGBoost
* Understanding performance differences

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* LightGBM
* XGBoost
* Jupyter Notebook

---

## ▶️ How to Run

1. Install required libraries:

   ```bash
   pip install numpy pandas scikit-learn lightgbm xgboost jupyter
   ```
2. Launch Jupyter Notebook:

   ```bash
   jupyter notebook
   ```
3. Open the notebook and run all cells
