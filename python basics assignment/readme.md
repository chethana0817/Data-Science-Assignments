🐍 Python Basics Assignment

This repository contains a collection of beginner-level Python programs designed to demonstrate fundamental programming concepts and problem-solving techniques.

---

📌 Project Overview

This assignment covers essential Python topics such as:

* Conditional statements
* Loops
* Functions
* Random number generation
* Basic real-world problem solving

---

📚 Exercises Included

### 🔹 Prime Number Checker

A program to check whether a given number is prime or not.

### 🔹 Product of Random Numbers

Generates two random numbers and calculates their product.

### 🔹 Even/Odd Square Logic

Processes numbers within a range and performs operations based on whether they are even or odd.

### 🔹 Hospital Billing System

A simple billing system that calculates total hospital expenses based on:

* Room charges
* Doctor consultation fees
* Lab test costs
* Medicine charges

---

🛠️ Technologies Used

* Python
* Jupyter Notebook

▶️ How to Run

1. Install Jupyter Notebook or use any Python IDE
2. Open the `.ipynb` file
3. Run the cells step by step
