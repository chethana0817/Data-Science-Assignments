# 🤖 Support Vector Machine (SVM) Project

This repository contains a Machine Learning project implementing the Support Vector Machine (SVM) algorithm for classification tasks.

---

## 📌 Project Overview

Support Vector Machine (SVM) is a powerful supervised machine learning algorithm used for classification and regression problems. It works by finding the optimal hyperplane that separates data into different classes.

This project demonstrates:

* Building an SVM model
* Training and testing the model
* Evaluating classification performance

---

## 📂 Dataset

The dataset used in this project contains labeled data used for training and evaluating the SVM model.

---

## 📚 Steps Performed

### 🔹 Data Preprocessing

* Handling missing values
* Encoding categorical variables
* Feature scaling (important for SVM)

### 🔹 Model Building

* Applying Support Vector Machine algorithm
* Training the model with dataset

### 🔹 Model Evaluation

* Predicting results
* Confusion Matrix
* Accuracy measurement

### 🔹 Model Optimization

* Understanding kernel functions
* Improving model performance
