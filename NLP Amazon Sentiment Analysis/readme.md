# 💬 NLP Amazon Sentiment Analysis Project

This repository contains a Natural Language Processing (NLP) project that performs sentiment analysis on Amazon reviews.

---

## 📌 Project Overview

Sentiment Analysis is a technique used to determine whether text data expresses positive, negative, or neutral sentiment.

This project demonstrates:

* Processing textual data
* Converting text into numerical format
* Building a sentiment classification model

---

## 📂 Dataset

The dataset used in this project contains Amazon product reviews along with sentiment labels.

---

## 📚 Steps Performed

### 🔹 Text Preprocessing

* Removing punctuation and stopwords
* Tokenization
* Text cleaning

### 🔹 Feature Extraction

* Bag of Words (BoW) / TF-IDF
* Converting text into numerical vectors

### 🔹 Model Building

* Training classification model
* Applying machine learning algorithms

### 🔹 Model Evaluation

* Predicting sentiments
* Confusion Matrix
* Accuracy evaluation

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* NLP libraries (NLTK / similar)
* Jupyter Notebook

---

## ▶️ How to Run

1. Install required libraries:

   ```bash
   pip install numpy pandas scikit-learn nltk jupyter
   ```
2. Launch Jupyter Notebook:

   ```bash
   jupyter notebook
   ```
3. Open the notebook and run all cells
