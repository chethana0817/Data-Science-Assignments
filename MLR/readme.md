# 🤖 Multiple Linear Regression (MLR) Project

This repository contains a Machine Learning project that implements Multiple Linear Regression to analyze relationships between variables and make predictions.

---

📌 Project Overview

Multiple Linear Regression (MLR) is a supervised learning algorithm used to model the relationship between one dependent variable and multiple independent variables.

This project demonstrates how to:

* Build a regression model
* Train and test the model
* Evaluate performance
* Interpret results

---

📂 Dataset

The dataset used in this project contains multiple input features that influence the target variable. It is used to train and evaluate the regression model.

---

📚 Steps Performed

### 🔹 Data Preprocessing

* Handling missing values
* Encoding categorical variables
* Splitting dataset into training and testing sets

### 🔹 Model Building

* Applying Multiple Linear Regression
* Training the model using training data

### 🔹 Model Evaluation

* Predicting test results
* Comparing predicted vs actual values
* Evaluating model accuracy

### 🔹 Feature Analysis

* Understanding impact of independent variables
* Interpreting coefficients

---
