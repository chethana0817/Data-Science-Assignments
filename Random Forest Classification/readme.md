# 🌲 Random Forest Classification Project

This repository contains a Machine Learning project implementing the Random Forest algorithm for classification tasks.

---

## 📌 Project Overview

Random Forest is an ensemble learning method that builds multiple decision trees and combines their outputs to improve accuracy and reduce overfitting.

This project demonstrates:

* Building a Random Forest model
* Training and testing the model
* Evaluating performance

---

## 📂 Dataset

The dataset used in this project contains labeled data used to train and evaluate the Random Forest model.

---

## 📚 Steps Performed

### 🔹 Data Preprocessing

* Handling missing values
* Encoding categorical variables
* Splitting dataset into training and testing sets

### 🔹 Model Building

* Applying Random Forest algorithm
* Training multiple decision trees

### 🔹 Model Evaluation

* Predicting results
* Confusion Matrix
* Accuracy calculation

### 🔹 Ensemble Learning

* Understanding how multiple models improve performance
* Reducing overfitting compared to single decision trees

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Jupyter Notebook

---

## ▶️ How to Run

1. Install required libraries:

   ```bash
   pip install numpy pandas scikit-learn jupyter
   ```
2. Launch Jupyter Notebook:

   ```bash
   jupyter notebook
   ```
3. Open the notebook and run all cells
